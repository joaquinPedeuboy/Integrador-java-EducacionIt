## Proyecto Integrador - Buscador de Peliculas

Una aplicacion que permite buscar peliculas por su genero o titulo y por su codigo. Ofrece tambien informacion sobre las peliculas.

## Tecnologias Aplicadas 
-Java
-Maven
-MySQL
-Git

## Version de Java
-Java 8
